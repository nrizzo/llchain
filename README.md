# clc-viz-experiments

```
Usage: clc-viz [-n ANCHORNUM] [-g gap-gap.bmp]
Simple visualization of optimal colinear chaining recursive scores

  -h, --help                    Print help and exit
  -V, --version                 Print version and exit
  -n, --anchors=ANCHORNUM       Number of anchors  (default=`3')
  -g, --gap-gap-output-file=BMPFILE
                                Visualize the gap-gap case in this file (BMP
                                  format)  (default=`gap-gap.bmp')
  -r, --random-seed=INT         Seed for the PRNG (-1 is different at every
                                  invocation)  (default=`-1')
```

## Dependencies
`gengetopt` for development
